========================================
Multi-LLM 블로그 자동화 시스템
빠른 시작 가이드
========================================

1단계: 패키지 설치
------------------
install.bat 파일을 실행하세요.

또는 수동으로:
- Frontend: npm install
- Backend: cd backend && pip install -r requirements.txt


2단계: 환경 변수 설정 (선택사항)
----------------------------------
backend/.env 파일을 생성하고 API 키를 설정하세요.

사용할 모델의 API 키만 설정하면 됩니다:
- OpenAI 사용 시: OPENAI_API_KEY
- Claude 사용 시: ANTHROPIC_API_KEY  
- Gemini 사용 시: GEMINI_API_KEY


3단계: 서버 시작
----------------
start.bat 파일을 실행하세요.

또는 수동으로:
- Backend: cd backend && python main.py
- Frontend: npm run dev


4단계: 브라우저에서 접속
------------------------
http://localhost:3000
