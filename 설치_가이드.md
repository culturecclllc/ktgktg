# 설치 가이드

## 필수 요구사항

### 1. Node.js 설치
- Node.js 18 이상 필요
- 다운로드: https://nodejs.org/
- 설치 후 터미널에서 확인:
  ```bash
  node --version
  npm --version
  ```

### 2. Python 설치
- Python 3.8 이상 필요
- 다운로드: https://www.python.org/
- 설치 시 "Add Python to PATH" 옵션 체크 필수!
- 설치 후 터미널에서 확인:
  ```bash
  python --version
  pip --version
  ```

## 설치 방법

### 방법 1: 자동 설치 (권장)

```bash
install.bat
```

이 스크립트가 Frontend와 Backend 패키지를 모두 설치합니다.

### 방법 2: 수동 설치

#### Frontend 설치
```bash
npm install
```

또는
```bash
install_frontend.bat
```

#### Backend 설치
```bash
cd backend
pip install -r requirements.txt
cd ..
```

또는
```bash
install_backend.bat
```

## 환경 변수 설정 (필수: Notion API 키)

**⚠️ 중요**: 다른 컴퓨터에서 클론한 경우, `backend/.env` 파일을 생성해야 합니다.

`backend/env.example` 파일을 참고하여 `backend/.env` 파일을 생성하세요:

**필수 설정 (Notion)**:
```env
# 로그인 인증용 Notion API 키 및 Database ID
NOTION_API_KEY=your_notion_login_api_key_here
NOTION_DATABASE_ID=your_notion_login_database_id_here

# 글 저장용 Notion API 키 및 Database ID
ARTICLE_NOTION_API_KEY=your_notion_article_api_key_here
ARTICLE_DATABASE_ID=your_notion_article_database_id_here
```

**선택 설정 (LLM API 키)**:
```env
# LLM API Keys (앱 내 설정 페이지에서도 입력 가능)
OPENAI_API_KEY=your_openai_api_key_here
GROQ_API_KEY=your_groq_api_key_here
GEMINI_API_KEY=your_gemini_api_key_here
```

## 문제 해결

### "ModuleNotFoundError: No module named 'fastapi'"
- Backend 패키지가 설치되지 않았습니다.
- `install_backend.bat` 실행 또는 `cd backend && pip install -r requirements.txt`

### "'next'은(는) 내부 또는 외부 명령"
- Frontend 패키지가 설치되지 않았습니다.
- `install_frontend.bat` 실행 또는 `npm install`

### "python이 인식되지 않습니다"
- Python이 PATH에 추가되지 않았습니다.
- Python 재설치 시 "Add Python to PATH" 옵션 체크
- 또는 Python 전체 경로 사용: `C:\Python3x\python.exe -m pip install ...`

### "npm이 인식되지 않습니다"
- Node.js가 설치되지 않았습니다.
- https://nodejs.org/ 에서 다운로드 및 설치
